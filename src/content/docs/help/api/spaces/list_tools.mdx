## Endpoint

| Property | Value |
| --- | --- |
| Type | `query` |
| Method | `GET` |
| Path | `/api/external/v1/spaces/list_tools` |
| Handler | `OperatelyWeb.Api.Spaces.ListTools` |

## Authentication

- Requires a valid API token.
- Read-only and full-access tokens can execute this query.
