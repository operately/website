## Endpoint

| Property | Value |
| --- | --- |
| Type | `query` |
| Method | `GET` |
| Path | `/api/external/v1/spaces/list_discussions` |
| Handler | `OperatelyWeb.Api.Spaces.ListDiscussions` |

## Authentication

- Requires a valid API token.
- Read-only and full-access tokens can execute this query.

## Outputs

| Field | Type | Required | Nullable |
| --- | --- | --- | --- |
| `discussions` | array of `discussion` object | Yes | No |
| `my_drafts` | array of `discussion` object | No | Yes |
