## Endpoint

| Property | Value |
| --- | --- |
| Type | `query` |
| Method | `GET` |
| Path | `/api/external/v1/goals/get` |
| Handler | `OperatelyWeb.Api.Goals.Get` |

## Authentication

- Requires a valid API token.
- Read-only and full-access tokens can execute this query.
