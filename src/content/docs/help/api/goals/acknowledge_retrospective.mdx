## Endpoint

| Property | Value |
| --- | --- |
| Type | `mutation` |
| Method | `POST` |
| Path | `/api/external/v1/goals/acknowledge_retrospective` |
| Handler | `OperatelyWeb.Api.Goals.AcknowledgeRetrospective` |

## Authentication

- Requires a valid API token.
- Requires a write-enabled token.
- Read-only tokens return `403`.
