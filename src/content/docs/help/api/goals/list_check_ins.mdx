## Endpoint

| Property | Value |
| --- | --- |
| Type | `query` |
| Method | `GET` |
| Path | `/api/external/v1/goals/list_check_ins` |
| Handler | `OperatelyWeb.Api.Queries.ListGoalCheckIns` |

## Authentication

- Requires a valid API token.
- Read-only and full-access tokens can execute this query.
