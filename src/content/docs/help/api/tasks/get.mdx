## Endpoint

| Property | Value |
| --- | --- |
| Type | `query` |
| Method | `GET` |
| Path | `/api/external/v1/tasks/get` |
| Handler | `OperatelyWeb.Api.Tasks.Get` |

## Authentication

- Requires a valid API token.
- Read-only and full-access tokens can execute this query.
