## Endpoint

| Property | Value |
| --- | --- |
| Type | `mutation` |
| Method | `POST` |
| Path | `/api/external/v1/tasks/update_milestone_and_ordering` |
| Handler | `OperatelyWeb.Api.Tasks.UpdateMilestoneAndOrdering` |

## Authentication

- Requires a valid API token.
- Requires a write-enabled token.
- Read-only tokens return `403`.
