## Endpoint

| Property | Value |
| --- | --- |
| Type | `mutation` |
| Method | `POST` |
| Path | `/api/external/v1/tasks/move` |
| Handler | `OperatelyWeb.Api.Tasks.Move` |

## Authentication

- Requires a valid API token.
- Requires a write-enabled token.
- Read-only tokens return `403`.
