## Endpoint

| Property | Value |
| --- | --- |
| Type | `query` |
| Method | `GET` |
| Path | `/api/external/v1/companies/list_activities` |
| Handler | `OperatelyWeb.Api.Companies.ListActivities` |

## Authentication

- Requires a valid API token.
- Read-only and full-access tokens can execute this query.
