## Endpoint

| Property | Value |
| --- | --- |
| Type | `query` |
| Method | `GET` |
| Path | `/api/external/v1/companies/get_work_map` |
| Handler | `OperatelyWeb.Api.Companies.GetWorkMap` |

## Authentication

- Requires a valid API token.
- Read-only and full-access tokens can execute this query.
