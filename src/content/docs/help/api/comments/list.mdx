## Endpoint

| Property | Value |
| --- | --- |
| Type | `query` |
| Method | `GET` |
| Path | `/api/external/v1/comments/list` |
| Handler | `OperatelyWeb.Api.Comments.List` |

## Authentication

- Requires a valid API token.
- Read-only and full-access tokens can execute this query.
