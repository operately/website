## Endpoint

| Property | Value |
| --- | --- |
| Type | `mutation` |
| Method | `POST` |
| Path | `/api/external/v1/links/create` |
| Handler | `OperatelyWeb.Api.Links.Create` |

## Authentication

- Requires a valid API token.
- Requires a write-enabled token.
- Read-only tokens return `403`.
