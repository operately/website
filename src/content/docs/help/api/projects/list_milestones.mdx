## Endpoint

| Property | Value |
| --- | --- |
| Type | `query` |
| Method | `GET` |
| Path | `/api/external/v1/projects/list_milestones` |
| Handler | `OperatelyWeb.Api.Projects.ListMilestones` |

## Authentication

- Requires a valid API token.
- Read-only and full-access tokens can execute this query.
