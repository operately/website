## Endpoint

| Property | Value |
| --- | --- |
| Type | `query` |
| Method | `GET` |
| Path | `/api/external/v1/projects/get_discussion` |
| Handler | `OperatelyWeb.Api.Projects.Discussions.Get` |

## Authentication

- Requires a valid API token.
- Read-only and full-access tokens can execute this query.
