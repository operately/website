## Endpoint

| Property | Value |
| --- | --- |
| Type | `mutation` |
| Method | `POST` |
| Path | `/api/external/v1/projects/close` |
| Handler | `OperatelyWeb.Api.Projects.Close` |

## Authentication

- Requires a valid API token.
- Requires a write-enabled token.
- Read-only tokens return `403`.
