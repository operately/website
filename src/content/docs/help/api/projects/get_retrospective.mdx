## Endpoint

| Property | Value |
| --- | --- |
| Type | `query` |
| Method | `GET` |
| Path | `/api/external/v1/projects/get_retrospective` |
| Handler | `OperatelyWeb.Api.Projects.GetRetrospective` |

## Authentication

- Requires a valid API token.
- Read-only and full-access tokens can execute this query.
