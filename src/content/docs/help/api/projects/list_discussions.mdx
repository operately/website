## Endpoint

| Property | Value |
| --- | --- |
| Type | `query` |
| Method | `GET` |
| Path | `/api/external/v1/projects/list_discussions` |
| Handler | `OperatelyWeb.Api.Projects.Discussions.List` |

## Authentication

- Requires a valid API token.
- Read-only and full-access tokens can execute this query.
