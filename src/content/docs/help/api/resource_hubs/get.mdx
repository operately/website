## Endpoint

| Property | Value |
| --- | --- |
| Type | `query` |
| Method | `GET` |
| Path | `/api/external/v1/resource_hubs/get` |
| Handler | `OperatelyWeb.Api.ResourceHubs.Get` |

## Authentication

- Requires a valid API token.
- Read-only and full-access tokens can execute this query.
