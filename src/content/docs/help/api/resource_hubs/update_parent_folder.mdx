## Endpoint

| Property | Value |
| --- | --- |
| Type | `mutation` |
| Method | `POST` |
| Path | `/api/external/v1/resource_hubs/update_parent_folder` |
| Handler | `OperatelyWeb.Api.ResourceHubs.UpdateParentFolder` |

## Authentication

- Requires a valid API token.
- Requires a write-enabled token.
- Read-only tokens return `403`.
