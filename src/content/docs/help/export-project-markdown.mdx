You can easily export all the information associated with a project into a Markdown file. This is useful for reporting, offline access, or sharing project details with external stakeholders.

## How to export a project

<Steps>
1. Navigate to the specific project page you want to export.
2. Locate the **Actions** menu in the right sidebar.
3. Click **Export as Markdown**.
</Steps>

The file will automatically download to your device containing all the project's information formatted in Markdown.
