Goals have a **Docs & Files** tab so supporting materials, such as files, documents and links, can be kept on the goal itself — not only on related projects.

## How to save content to a goal

<Steps>
1. Open a goal and go to the **Docs & Files** tab.
2. Click **Add**.
3. Choose one of:
   - **New document**
   - **New folder**
   - **Upload files**
   - **Add link**
4. Fill out the form for the type you chose.
5. Return later to open drafts or published items in the same goal context.
</Steps>

Activity for docs and files on goals appears in the goal's activity feed.
