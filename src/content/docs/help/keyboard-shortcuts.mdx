Operately supports keyboard shortcuts for global navigation and for working with tasks in list and board views.

## Open the cheatsheet

- **Help** menu → **Keyboard shortcuts**
- Press **?** from anywhere in the app

The **Keyboard Shortcuts Cheatsheet** lists shortcuts grouped by context.

## Global shortcuts

| Shortcut | Action |
| -------- | ------ |
| **⌘ K** (Mac) / **Ctrl K** (Windows) | **Open global search** |
| **?** | **Open keyboard shortcuts** |

See [Global Search](/help/global-search) for how search works.

## Task shortcuts

Use these when a task list or board supports keyboard selection:

| Shortcut | Action |
| -------- | ------ |
| **j** | **Start task selection or select next task** |
| **k** | **Select previous task** |
| **Return** | **Open selected task** |
| **a** | **Open assignee picker for the selected task** |
| **s** | **Open status picker for the selected task where supported** |
| **d** | **Open due date picker for the selected task** |
| **Esc** | **Clear task selection** |

<Aside>
Some shortcuts only work when a task is currently selected or when the current view supports inline editing. For full task editing in the slide-in, see [Edit a task](/help/kanban-board-edit-task).
</Aside>
