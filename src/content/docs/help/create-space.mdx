---
title: "Create a Space"
description: "Create a new space to organize work around a team or department."
---

import { Steps } from '@astrojs/starlight/components';

Anyone in the organization can create a new [space](/help/intro-to-spaces).

To create a new space:

<Steps>
1. Go to your organization Home.
2. On the right, above the grid of spaces, click **Add Space** button.
3. Enter a name for the space.
4. Optionally, write the purpose of the team or department.
5. Optionally, adjust the [access level](/help/space-access-control).
6. Click **Create Space**.
</Steps>
