Task reminders notify assignees at the right time. Configure them from the task sidebar on project tasks, milestone tasks, space-level tasks, and other task views that open the task slide-in.

## Prerequisites

The **Reminders** section appears only when the task has at least one assignee. If the task is unassigned, use **Assign task** first. Reminder controls also depend on whether you have permissions to edit the task.

## Set a reminder

<Steps>
1. Open a task from a Kanban board, space task list, project task list, or milestone task list.
2. In the sidebar, find **Reminders**.
3. Click **Add reminder**.
4. Choose a reminder type:
   - **Before due date**
   - **Due date**
   - **Overdue**
   - **On date**
5. Configure the option and save the reminder.
</Steps>

The default options shown can depend on whether the task already has a due date.

<Aside>
Reminder editing is only available when the task is in a state that supports reminders.
</Aside>
