You can easily export all the information associated with a goal into a Markdown file. This is useful for archiving and sharing the goal outside of Operately.

## How to export a goal

<Steps>
1. Navigate to the specific goal page you want to export.
2. Locate the **Actions** menu in the right sidebar.
3. Click **Export as Markdown**.
</Steps>

The file will automatically download to your device containing all the goal's information formatted in Markdown.
