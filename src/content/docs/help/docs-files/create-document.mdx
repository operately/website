description: "Create a new document to share information with your team."
---

import { Steps } from '@astrojs/starlight/components';

Anyone in a space can create a new document.

To create a new document:

<Steps>
1. Open the space where you want to create the document.
2. Click on the **Documents & Files** tool.
2. Click the **Add** button in the top left corner.
3. Select **New Document**.
4. Give your document a name and add your content.
5. Click **Submit** to save your document.
</Steps>

Members of the space will be notified that a new file was added.
