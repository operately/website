Anyone in the space who has [edit permissions](/help/space-access-control) can edit a document.

To edit a document:

<Steps>
1. Open the document you want to edit.
2. Click on the **...** menu in the top right corner.
3. Select **Edit** from the dropdown menu.
4. Make your changes to the document.
5. Click **Save** to save your changes.
</Steps>
