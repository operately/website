Create folders to organize the content in your space.

To create a folder:

<Steps>
1. Open the space where you want to create the folder.
2. Click on the **Documents & Files** tool.
3. Click the **Add** button in the top left corner.
4. Select **New Folder**.
5. Give your folder a name.
6. Click **Save** to create the folder.
</Steps>

Anyone in a space can create folders to organize content. Folders can 
contain documents, files, and other folders.
