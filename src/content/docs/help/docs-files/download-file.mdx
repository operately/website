Anyone in a space can download files to their computer.

To download a file:

<Steps>
1. Open the space where the file is located.
2. Click on the **Documents & Files** tool.
3. Click on the file you want to download.
4. Click on the **...** menu in the top right corner.
5. Select **Download** from the dropdown menu.
6. The file will be downloaded to your computer.
</Steps>
