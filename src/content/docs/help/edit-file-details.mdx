Anyone in the space who has [edit permissions](/help/space-access-control) can edit a file's title or description.

To edit a file's title or description:

<Steps>
1. Open the file you want to edit.
2. Click on the **...** menu in the top right corner.
3. Select **Edit** from the dropdown menu.
4. Make your changes to the file's title or description.
5. Click **Save** to save your changes.
</Steps>
