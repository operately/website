Projects have a **Docs & Files** tab so supporting materials, such as files, documents and links, can be kept on the project itself.

## How to save content to a project

<Steps>
1. Open a project and go to the **Docs & Files** tab.
2. Click **Add**.
3. Choose one of:
   - **New document**
   - **New folder**
   - **Upload files**
   - **Add link**
4. Fill out the form for the type you chose.
5. Return later to open drafts or published items in the same project context.
</Steps>

Activity for docs and files on projects appears in the project's activity feed.
